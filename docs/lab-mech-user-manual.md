# 機械性質試驗紀錄表 操作說明

## 1. 功能用途

本功能用來建立、查詢、修改、匯出「機械性質試驗紀錄表」。

本頁面負責：
- 建立新報告
- 載入既有報告
- 修改測試內容
- 匯出 DOCX
- 軟刪除報告

本頁面**不負責總判定**。  
總判定請到外部功能：
- `機械性質試驗紀錄表判定`

## 2. 進入方式

由左側選單進入：
- `實驗室管理`
- `機械性質試驗紀錄表`

## 3. 上方按鈕說明

- `選擇委託單`
  從既有委託單帶入基本資料。

- `新建報告`
  依模板建立一份新的機械性質試驗紀錄表。

- `載入報告`
  開啟搜尋視窗，載入既有報告。

- `查修`
  進入「機械性質試驗紀錄表查修」頁面，可查詢、修改或刪除既有報告。

- `重設`
  清空目前頁面資料，但會重新套用預設值，例如登入者與今日日期。

- `複製成新報告`
  以目前報告為基礎建立一份新報告，可保留原資料快速另存。

- `全部判定`
  只對目前頁面中的各測試項目做自動判定計算，方便檢查各項是否合格。
  這不是最終總判定。

- `匯出 DOCX`
  匯出目前報告為 Word 檔。

- `使用說明`
  開啟本功能的操作說明。

## 4. 基本資料

### 4.1 由委託單自動帶入的欄位

使用 `選擇委託單` 後，系統會盡量帶入：
- 委託單編號
- 客戶名稱
- 日期
- 聯絡人
- 電話
- E-mail
- 地址
- 品名
- 規格
- 材質編號
- 批號
- 批量
- 電鍍別
- 製造廠商
- 圖號或法規標準

### 4.2 預設值

新建或重設時，系統會預設：
- `測試人員` = 目前登入者
- `測試日期` = 今日
- `完成日期` = 今日

以上 3 個欄位都可以手動修改。

### 4.3 來源表單

內部 `來源表單` ID 已隱藏，不顯示在畫面上。

## 5. 試驗方法與儀器編號

目前畫面上：
- `心部硬度`
- `表面硬度`

這兩段的：
- `試驗方法`
- `儀器編號`

都已改成共用清單模式。

### 5.1 可直接選取

可由下拉選單選擇既有資料。

### 5.2 可直接輸入

如果清單裡沒有，也可以直接輸入新值。  
輸入後系統會自動加入共用清單，之後可直接選。

### 5.3 點 `+` 維護清單

在 `試驗方法` 或 `儀器編號` 右側點 `+`：
- 可新增新項目
- 可點既有項目後修改名稱

## 6. 心部硬度 / 表面硬度的上下限用途

`下限`、`上限` 是規格區間，用來做自動判定。

判定邏輯：
1. 每片試片先輸入 3 個量測值
2. 系統先算平均值
3. 再用「四捨六入，五成雙」得到判定值
4. 用判定值去比較下限與上限
5. 區間內為 `PASS`，超出為 `FAIL`

例如：
- 平均值 `24.5` 會判成 `24`
- 平均值 `25.5` 會判成 `26`

## 7. 載入既有報告

點 `載入報告` 後，可用以下條件搜尋：
- 報告編號
- 委託單編號
- 客戶名稱

按 `搜尋` 後，清單會列出符合條件的報告。  
按該筆的 `載入` 即可開啟。

## 8. 查修

點 `查修` 可進入查修頁面。

查修頁面用途：
- 查詢既有報告
- 點報告編號進入修改
- 單筆刪除或批次刪除

刪除規則：
- 不是真正刪除資料
- 系統只會做 `is_deleted = 1` 的軟刪除
- 一般使用者之後不會再看到這筆資料

## 9. 判定作業

機械性質試驗紀錄表的最終判定，不在本頁做。

請到：
- `機械性質試驗紀錄表判定`

由該頁統一進行：
- 判定結果選擇
- 審查人員寫入
- 判定日期寫入

## 10. 匯出 DOCX

按 `匯出 DOCX` 可輸出 Word 檔。  
匯出前建議先：
- 確認基本資料完整
- 確認測試值已輸入
- 確認需要的項目都已判定

## 11. 常見操作流程

### 11.1 從委託單建立新報告

1. 點 `選擇委託單`
2. 搜尋並選取來源委託單
3. 檢查基本資料是否正確帶入
4. 補齊各測試項目的數值
5. 按 `全部判定` 檢查各項結果
6. 按 `儲存`
7. 需要時按 `匯出 DOCX`

### 11.2 載入既有報告修改

1. 點 `載入報告`
2. 以報告編號、委託單編號或客戶名稱搜尋
3. 點 `載入`
4. 修改內容
5. 按 `儲存`

### 11.3 快速維護試驗方法或儀器編號

1. 在 `心部硬度` 或 `表面硬度` 區塊
2. 點欄位右側 `+`
3. 直接輸入新名稱新增
4. 或先點既有項目，再修改名稱

## 12. 注意事項

- 本頁沒有總判定顯示，也不負責最終總判定。
- 最終總判定請由 `機械性質試驗紀錄表判定` 功能處理。
- 新增、修改、刪除都會記錄執行者。
- 刪除為軟刪除，不會真的把資料從資料庫移除。
